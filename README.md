# CRM Monorepo Foundation

Production-grade CRM starter organized for scale with:

- `backend/`: Express + TypeScript + MongoDB foundation
- `frontend/`: Next.js App Router + TypeScript dashboard shell
- `docs/`: architecture, standards, and conventions

## Quick start

### Backend

```bash
cd backend
npm install
npm run dev
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

## Scope

This scaffold intentionally excludes business logic. It provides structure, configuration, middleware, and layout primitives only.
